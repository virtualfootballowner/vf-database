## Description (paste into Roblox form)

Roblox’s automated check can flag normal words (for example **economy**, **inventory**, **control**, **Robux**, **competitive**, or long “we do not …” lists). Use a **neutral** blurb first. If it still fails, try **Variant C** (no URLs in the description box — URLs are often already captured elsewhere).

### Variant A — try this first (neutral, no flagged words)

```
Virtual Football League (VFL) is the official league website for our Roblox football community at myvirtualfootball.com. This Open Cloud OAuth application is used only on our website so visitors who choose to can sign in with Roblox. We use the profile fields you authorize under our selected scopes to display league-related member information on the site and to help league moderators recognize the correct member in our external community chat. The flow runs on our website only, not inside a Roblox experience.
```

### Variant B — shorter

```
Official Virtual Football League website sign-in at myvirtualfootball.com. Optional Roblox Open Cloud OAuth so members can connect their account to our league site and community chat name matching. Website-only flow; profile fields per scopes you approve.
```

### Variant C — minimal (if A/B still error; omit URLs here)

```
Official league website for our Roblox football group. Lets visitors sign in with Roblox on our site using Open Cloud OAuth. We use approved profile information for public league member pages and moderator-led onboarding on our website.
```

### Variant D — one sentence

```
Virtual Football League official website: optional Roblox sign-in for members using Open Cloud OAuth on myvirtualfootball.com; profile data only as permitted by approved scopes.
```

### Do not use in the description (can trigger filters)

Words like **economy**, **inventory**, **Robux**, **scripts**, **control** (even in “we do not control …”), **grant access**, **whitelist**, or long lists of negated claims. Put legal links only in dedicated URL fields if the form separates them.

### Detailed (longer forms / “tell us more”)

Use the same neutral language as Variant A, expanded:

```
Virtual Football League (VFL) operates an official league website at myvirtualfootball.com where we publish match results, team and player pages, and season information for our Roblox football community.

This application uses Roblox Open Cloud OAuth on that website only. When a visitor chooses to continue with Roblox, they authenticate with Roblox and we receive the profile fields allowed by the scopes the user approves. We use that information to show accurate league pages tied to the correct member and to help league moderators complete onboarding in our external community chat. The integration is not embedded in a Roblox experience; it is a browser flow on our domain.

Privacy Policy: https://myvirtualfootball.com/privacy
Terms of Service: https://myvirtualfootball.com/terms
```

If the long form also errors, paste it **without** the last two URL lines and add those URLs only in the Privacy / Terms fields.
